# 🐍 Snake Game (Python)

A classic Snake game built in Python. Control the snake, eat food to grow longer, and avoid colliding with the walls or your own tail. This project helped me practice core programming concepts like game loops, collision detection, and state management.

---

## ✅ Features

- Smooth snake movement
- Food spawns randomly on the screen
- Score tracking
- Game Over when:
  - Snake hits the wall
  - Snake hits its own body

---

## 🎮 Controls

- **Up Arrow**: Move Up
- **Down Arrow**: Move Down
- **Left Arrow**: Move Left
- **Right Arrow**: Move Right

---

## 🧱 Tech Stack

- **Python**
- **turtle** (Python standard library)

---

## ▶️ How to Run

### 1) Clone the repo / open the project folder

Make sure you are inside the Snake project directory.

### 2) Run the game

```bash
python main.py
```
